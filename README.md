# SPOORTHI - Technical Symposium Website

Official website for **SPOORTHI**, the National Level Technical Symposium organized by the Department of Electronics and Communication Engineering, JNTUH University College of Engineering, Science and Technology Hyderabad.

## Overview

SPOORTHI is an annual technical fest that brings together students from across the nation to participate in various technical, cultural, gaming, and workshop events. This website serves as the primary platform for event information, registrations, and fest updates.
